# Forbidden

- Comments, docstrings
- Error handling (try-except)
- Retry, timeout logic
- Hardcoded values
- Mocking
- Root file creation
- Modifying `data/prompts.yaml` without explicit user instruction (PROTECTED FILE)

# Required

- Config in yaml/.env
- Type hints on all functions
- Small, composable functions
- Short files, isolated concerns

# Tools

- Python: uv, ruff
- Tasks: Taskfile.yaml
- Frontend: npm

# Model

Use gemini-3-flash-preview, always in config.yaml
